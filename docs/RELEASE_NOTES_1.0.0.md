# OdontoCare 1.0.0

Fecha de compilacion: 7 de junio de 2026.

## Incluye

- Gestion de pacientes e historial clinico.
- Odontograma interactivo con historial por pieza.
- Tratamientos, pagos, abonos y saldos.
- Agenda diaria, semanal y mensual con reprogramacion.
- Imagenes, radiografias y PDF almacenados localmente.
- Inventario, proveedores y movimientos.
- Dashboard, reportes, busqueda global y auditoria.
- Usuarios con roles de administrador, odontologo y recepcion.
- PostgreSQL portable, migraciones automaticas y funcionamiento offline.
- Backups automaticos, manuales, importacion y restauracion.
